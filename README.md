# Qualcomm Sensing Hub

Qualcomm Sensing Hub (QSH) is an always-on, low-power interface designed to collect, process, and combine sensor and contextual data. It offers stable APIs that enable applications and services to interact with the hub efficiently, reducing the need for frequent application processor wakeups.
